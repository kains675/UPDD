### Strategic Positioning

UPDD positions itself within the ncAA peptide drug discovery landscape with a deliberately bounded scope:

- **ncAA-first**: The pipeline's distinguishing focus is ncAA-modified peptide complexes, where standard force-field workflows have known parameterization gaps. UPDD's Stage 4 evaluation framework (MD + QM/MM + MM-PBSA) is designed around honest handling of these gaps rather than masking them.
- **Level 1 (current scope)**: A decision support tool — UPDD aids the design–evaluate cycle for ncAA peptides but does not yet claim quantitative experimental match. Capability Levels 1/2/3 are defined in §4.1.1.
- **Branched ΔΔG core**: The methodology centers on per-variant branched ΔΔG calculations seeded from a parent design, enabling direct relative comparison of ncAA modifications under a shared MD ensemble (see §4.1.3 for the iterative cycle architecture).
- **Wet-lab-integrated**: Iterative cycle architecture (Cycle 0 baseline → Cycle 1+ refinement) is structured so each computational round can ingest experimental feedback without disturbing the upstream sampling state.

This positioning is conservative by design: the v0.7 release is a Level 1 milestone, not a quantitative-match release.
