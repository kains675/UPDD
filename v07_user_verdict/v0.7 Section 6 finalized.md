## 6. Phase β + γ Roadmap

The post-Phase α roadmap consists of two scientifically distinct phases plus a conditional refinement phase. Each phase has a specific entry criterion, scope, and expected outcome — designed to enable measurable progress assessment.
v0.7 (now) → Phase β → v0.7.1 → Phase γ → v0.8 → Phase γ' (conditional) → v0.9
(multi-system)    (Khoury RESP)      (methodology refinement)
~2-4 weeks         ~2-3 months        ~6+ months

### 6.4 Phase γ' — Methodology Refinement (v0.9, conditional)

**Triggered if**: Phase γ residual gap > 2 kcal/mol on Cp4 (likely per §3.10).

**Candidate refinements** (in order of decreasing field-readiness):

1. **Explicit solvent + entropy correction** (Hou 2011 multi-MD sampling)
   - Cost: ~2× compute per system
   - Expected improvement: 0.5-1 kcal/mol
   - Maturity: established, multiple groups using

2. **Polarizable force fields** (AMOEBA family, Ren 2002)
   - Cost: ~10× compute
   - Expected improvement: 1-2 kcal/mol (case-dependent)
   - Maturity: emerging, requires AMOEBA expertise

3. **QM/MM at binding site**
   - Cost: ~100× compute
   - Expected improvement: case-by-case (potentially significant)
   - Maturity: established but specialized

4. **Alchemical free energy** (TI/FEP)
   - Cost: comparable to MM-PBSA but different statistical framework
   - Expected improvement: depends on sampling
   - Maturity: established, but reweighting concerns for cyclic peptides

#### 6.4.1 Phase γ' Decision Process

After Phase γ completion:
- Measure residual gap (Cp4 + multi-system)
- Assess which candidate addresses the dominant residual axis
- May require multiple γ' iterations
- v0.9 release may reflect partial implementation of selected candidate(s)

#### 6.4.2 Phase γ' Estimated Timeline
Highly variable: 6+ months (low confidence)

Depends on candidate selection
Depends on field state of selected methods
May be deferred indefinitely if Phase γ acceptable


### 6.5 Phase β / γ Risk Mitigation

**Risk: Phase β scope creep**
- Mitigation: Strict Definition 3 criterion (≥3 systems sign-significance)
- Strict timeline (2-4 weeks max for v0.7.1)
- Defer additional systems to Phase γ if needed

**Risk: PR-6 R&D longer than estimated**
- Mitigation: Existing-data Phase β stage runs in parallel with PR-6 development
- v0.7.1 path stays on track even if PR-6 delayed
- N-methyl systems can be deferred to v0.7.2 if necessary

**Risk: Phase γ FF mixing artifacts**
- Mitigation: SciVal pre-validation cycle required (V1-style multi-axis assessment)
- Comparison with Phase α charge fix as architectural baseline
- Honest acknowledgment if FF mixing unsuccessful

**Risk: Khoury R²=0.388 ceiling makes Phase γ less impactful**
- Mitigation: Phase γ value is methodology change, not just accuracy
- Even partial improvement is publishable
- Phase γ' candidates ready as next axis

### 6.6 Multi-Phase Coordination

**Cross-phase dependencies**:
- v0.7.1 release: PR-6 (HIGH) + Phase β multi-system data
- v0.8 release: PR-18 (HIGH) + Phase γ Khoury integration
- v0.9 release: Phase γ' candidate selection + implementation

**Cross-phase patterns**:
- Each phase has explicit entry criterion (process or outcome)
- Each phase has SAFE/RISKY/FAIL verdict possibility
- Each phase produces v-number release with honest disclosure
- Each phase informs the next via measurable evidence
